Colabore

Que bom que você se interessou em colaborar com o projeto! :) Existem algumas maneiras possíveis:

Tecnicamente

Desenvolver e manter a plataforma e novos scripts de libertação de dados é de suma importância para que o projeto seja bem sucedido. Entre em nosso chat para conversarmos sobre a melhor maneira de colaborar.

  1. Utilizando e citando os dados disponíveis no site;
  2. Sugerindo um novo dataset para libertarmos;
  3. Libertando dados e o código do software.
Financeiramente

Possuímos diversos gastos para manter o site online e desenvolver novas funcionalidades e você pode também contribuir para a sustentabilidade financeira do projeto. Se tiver interesse, entre em contato.

  1. Encomendando a libertação de um novo dataset;
  2. Encomendando análises e visualizações complexas nos dados que já temos disponíveis;
  3. Doando recorrentemente.
